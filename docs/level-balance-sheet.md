# 레벨업 증가량 시트

현재 레벨업 카드가 보여주는 "증가량" 기준으로 정리한 밸런스 시트다.
기본 수치가 아니라, 해당 레벨업 1회를 선택했을 때 얼마나 늘어나는지만 적었다.

## 플레이어 레벨 XP 구간

공식: `다음 레벨 필요 XP = 5 + ((현재 레벨 - 1) * 3)`

| 현재 레벨 | 다음 레벨 필요 XP | 직전 대비 증가 |
| --- | ---: | ---: |
| 1 | 5 | +0 |
| 2 | 8 | +3 |
| 3 | 11 | +3 |
| 4 | 14 | +3 |
| 5 | 17 | +3 |
| 6 | 20 | +3 |
| 7 | 23 | +3 |
| 8 | 26 | +3 |
| 9 | 29 | +3 |
| 10 | 32 | +3 |

## 스탯 업그레이드 증가량

| 스탯 | 1회 선택 증가량 | 최대 레벨 |
| --- | --- | ---: |
| 공격력 | 피해 `+10%` | 5 |
| 공격속도 | 공속 `+5%` | 5 |
| 최대체력 | 최대체력 `+20` | 5 |
| 체력재생 | 초당 재생 `+0.5` | 5 |
| 이동속도 | 이동속도 `+6%` | 5 |
| 사거리 | 사거리 `+10%` | 5 |

## 지금 바로 봐야 할 밸런스 포인트

- 라이플 / 카타나 / 위성빔은 `LV4 -> LV5`에서 추가 효과를 얻는 대신 피해가 `-10%` 내려가고, `LV5 -> LV6`에서 피해 `+15%`로 회복된다.
- SMG / 샷건 / 체인 / 드론 / 포탑은 `LV4 -> LV5`의 피해 급증을 줄이고, `LV4 -> LV5`와 `LV5 -> LV6`에 각각 `+15%`씩 나눠 들어가도록 조정했다.
- 스나이퍼는 의도적으로 `LV4 -> LV5`, `LV9 -> LV10`의 상승폭을 크게 유지하고 있다.
- 플레이어 레벨 필요 XP는 끝까지 선형 `+3` 증가라, 후반 레벨업 빈도가 급격히 줄지는 않는다.
